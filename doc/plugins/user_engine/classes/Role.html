Class Role
In: vendor/plugins/user_engine/app/models/role.rb
Parent: ActiveRecord::Base

The Role class represents an abstract set of allowable behaviours within an application. Each Role is associated with a number of Permissions (or controller/action objects), and such associations indicate what actions users using this application are allowed to perform.
